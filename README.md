# 🏰 AI Tower Defence

Claude AI와 대결하는 실시간 전략 타워 디펜스 게임

---

## 📖 게임 소개

**AI Tower Defence**는 플레이어가 구조물을 건설하고 유닛을 생산하여 상대방 AI의 기지를 파괴하는 전략 게임입니다. Claude API를 연동하면 AI가 전략적으로 플레이하며, API 없이도 기본 AI로 플레이할 수 있습니다.

---

## 🎯 게임 목표

- **승리 조건**: 상대방(AI) 기지를 파괴
- **패배 조건**: 내 기지가 파괴됨

---

## 🎮 게임 방법

### 기본 흐름
1. 구조물을 건설하여 경제/군사력 확보
2. 병영에서 유닛 생산
3. **Start Round** 버튼으로 라운드 시작
4. 유닛들이 자동으로 길을 따라 이동하며 전투
5. 라운드 종료 후 자원 획득, 다음 라운드 준비

### 조작법
| 조작 | 설명 |
|------|------|
| 구조물 건설 | 우측 메뉴에서 선택 → 맵 클릭 |
| 구조물 선택 | 맵의 구조물 클릭 |
| 유닛 생산 | 병영 선택 → 유닛 버튼 클릭 |
| 구조물 판매 | 구조물 선택 → 판매 버튼 (50% 환불) |
| 취소 | ESC 키 |

---

## 🏗️ 구조물

| 구조물 | 비용 | 효과 |
|--------|------|------|
| 🏠 병영 | 20 | 유닛을 생산합니다 |
| 🏘️ 주거지 | 20 | 최대 인구수 +3 |
| 🗼 포탑 | 20 | 길 위의 적을 자동 공격 |
| ⛏️ 자원생산소 | 30 | 라운드당 자원 +20 |

---

## ⚔️ 유닛

| 유닛 | 비용 | 인구 | 체력 | 공격력 | 사거리 | 특징 |
|------|------|------|------|--------|--------|------|
| ⚔️ 근접 | 5 | 1 | 20 | 4 | 1 | 균형잡힌 기본 유닛 |
| 🏹 원거리 | 5 | 1 | 13 | 6 | 2.3 | 높은 화력, 낮은 체력 |
| 🛡️ 탱크 | 10 | 1 | 60 | 2 | 1 | 매우 높은 체력, 느린 속도 |

### 💡 전략 팁
- 탱크를 앞세우고 원거리가 뒤에서 공격하는 조합이 효과적!
- 포탑은 길 근처에 배치해야 효과적
- 초반에는 자원생산소로 경제력 확보

---

## ⚙️ 시스템

### 자원
- 매 라운드 시작 시 **50** 자원 지급
- 자원생산소 1개당 **+20** 추가 지급

### 인구수
- 유닛 생산에 필요
- 기본 최대 인구: **3**
- 주거지 1개당 **+3** 증가

### 구조물 제한
- 최대 **10개**까지 건설 가능

---

## 🤖 AI 모드

### Claude API 연동
1. 게임 시작 시 API 키 입력 또는 상단 **API 키 설정** 버튼 클릭
2. Anthropic Claude API 키 입력
3. AI가 매 라운드 전략적으로 구조물 건설 및 유닛 생산

### 기본 AI
- API 키 없이도 플레이 가능
- 라운드에 따라 자동으로 유닛 생성

---

- **Frontend**: HTML5 Canvas, Vanilla JavaScript, CSS3
- **AI**: Claude API (Anthropic)
